# Codex 桌面版 26.616.41845 macOS / Windows 对照

本批次在 macOS 样本中没有内容变化；Windows 样本显示主程序与内置插件版本推进，并带来任务、设置、调试和浏览器相关界面的可见信号。

## 关于这份报告

本文只依据两份官方签名分发包逆向重建后的机器提取事实。macOS 对比范围是 `26.616.41845 (build 4198)` 到同一版本同一 build；Windows 对比范围是 `26.616.4196.0` 到 `26.616.5445.0`。

平台归属使用 `[macOS+Windows]`、`[仅 macOS]`、`[仅 Windows]`。webview/assets 与 .vite/build 的 hash 改名按输入说明视为构建噪音，不作为功能增减依据。

### 【实证】[仅 Windows] 内置插件版本推进到 macOS short 对齐版本

Windows 包内三个 OpenAI bundled 插件的 `plugin.json` 版本从 `26.616.32156` 更新到 `26.616.41845`：

其中 `computer-use` 的描述仍指向通过 Computer Use 控制 Windows 桌面应用；`chrome` 描述仍覆盖基于用户现有 Chrome 状态的自动化；`browser` 描述仍覆盖 Codex 内置浏览器、本地目标导航、检查、点击、输入和截图等场景。事实包只显示版本字段变化，没有显示这些描述文本本身发生变化。
