这不是官方变更日志，而是一份**基于官方分发产物的逆向差异报告**。本次为历史回填：两个版本均超出官方 Sparkle(应用自动更新框架)增量窗口，因此采用两个官方全量包直接比较，而不是增量包重建。

对比范围为 `26.228.1430`(build 760，2026-03-01) → `26.303.1606`(build 806，2026-03-04)，平台为 macOS(macOS 操作系统) arm64(苹果芯片架构)。

可信度方面，两个官方全量包均有 SHA-256(哈希校验值)记录；公证验证结果均返回 `CSSMERR_TP_CERT_REVOKED`。事实包说明这是部分早期版本签名证书后来被吊销导致的结果，影响系统安全安装判断，但不影响对 bundle(应用包目录)内容做差异分析。

证据等级分两档：

- **【实证】**：来自 Info.plist(应用元数据文件)、package.json(项目依赖与脚本配置文件)、可读 diff(差异文本)、签名或校验结果等直接证据。
- **【信号】**：来自前端模块名或二进制体积变化等间接证据；这类信息只指向某方向，但无法从产物还原到源码级意图。

## ✨ 重点变化

### 1. 版本与构建号更新到 `26.303.1606` / build 806 【实证】

应用版本从 `26.228.1430` 更新到 `26.303.1606`，构建号从 `760` 更新到 `806`。这意味着这是 2026-03-04 发布的 macOS arm64 新构建。

关键证据：

```diff
- "version": "26.228.1430"
+ "version": "26.303.1606"

- "codexBuildNumber": "760"
+ "codexBuildNumber": "806"
```

Info.plist 中也同步更新：

```diff
/CFBundleShortVersionString: 26.228.1430 → 26.303.1606
/CFBundleVersion: 760 → 806
```

### 2. 新增开发元数据探测脚本入口 【实证】

`package.json` 新增了两个脚本：`metadata-path` 与 `metadata-probe`。从命名和命令看，它们都指向 `scripts/dev-metadata.ts`，用于查询或探测开发元数据路径。

这对普通用户未必可见，但对开发、调试或内部诊断流程是明确变化。

关键证据：

```diff
+ "metadata-path": "pnpm exec tsx ./scripts/dev-metadata.ts path",
+ "metadata-probe": "pnpm exec tsx ./scripts/dev-metadata.ts probe",
```

### 3. Playwright 代理调试脚本改为通过 `tsx` 预加载运行 【实证】

`playwright:agent:repl` 脚本从直接运行 Node.js(Node.js 运行时)脚本，改为使用 `node --import tsx` 方式运行。Playwright(浏览器自动化工具)相关脚本仍然存在，但启动方式发生变化。

这通常影响的是开发者调试 Electron(Electron 桌面应用框架)代理或 CDP(Chrome DevTools Protocol，浏览器调试协议)连接时的运行路径；事实包不能进一步证明用户侧行为变化。

关键证据：

```diff
- "playwright:agent:repl": "pnpm run rebuild:sqlite && node ./scripts/playwright-electron-agent-cdp.mjs",
+ "playwright:agent:repl": "pnpm run rebuild:sqlite && node --import tsx ./scripts/playwright-electron-agent-cdp.mjs",
```

### 4. 远程连接设置相关前端模块被移除 【信号】

去除 Vite(前端构建工具) hash(构建指纹)噪音后，事实包显示真实移除的前端模块为 `remote-connections-settings`。

这指向“远程连接设置”相关界面或逻辑在本版发生收缩、迁移或关闭，但无法从产物还原到源码级意图，也不能证明对应功能已经对用户删除。

关键证据：

```text
移除模块: remote-connections-settings
```

### 5. 主程序二进制体积明显下降，应用层包略增 【信号】

`Contents/Resources/codex` 从 84.4 MB 降至 77.1 MB，变化为 -7.37 MB；`Contents/Resources/app.asar` 从 36.4 MB 增至 36.5 MB，变化为 +0.12 MB。

这指向本版在原生主程序或打包内容上有调整，但体积变化只能说明产物变化，无法从产物还原到源码级意图。
