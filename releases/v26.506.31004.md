# Codex 桌面版 `26.506.31004`(build 2604)

这一版从 `26.506.21252` 到 `26.506.31004` 的变化较克制：bundle(应用包)文件数量不变，主要变化集中在 `app.asar`(Electron 应用资源包)里的少量前端模块与版本元数据。
