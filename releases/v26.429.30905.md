## 关于这份报告

这不是官方变更日志，而是一份**基于官方分发产物的逆向差异报告**。本次属于历史回填：两个版本均已超出官方 Sparkle(桌面应用自动更新框架) 增量窗口，因此没有使用增量包重建流程，而是下载两个官方全量包后直接做结构化 bundle(应用包) 差异。

可信度方面，本次信任根来自 Apple(苹果) 公证与 OpenAI Developer ID(开发者身份) 代码签名。上一版与最新版均通过 `spctl` 公证验证，结果为 `accepted`，来源为 `Notarized Developer ID`，签名来源均为 `Developer ID Application: OpenAI OpCo, LLC (2DC432GLL2)`。

本文使用两档证据等级：

- **【实证】**：来自人类可读的直接证据，例如 `Info.plist` 键值、签名验证结果、文件统计、依赖清单。
- **【信号】**：只有间接证据，例如前端模块名或二进制体积变化。它只能指向某方向，但无法从产物还原到源码级意图。

这意味着 macOS 系统、更新器与应用自身可识别为一个新的正式构建，但事实包中没有提供官方 release notes(发布说明)，也没有显示新的权限项或新增组件。

因此，本报告不把这些带 hash 的文件名变化写成“新增/删除了功能”。
