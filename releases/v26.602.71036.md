```diff

/CFBundleVersion: '3593' → '3685'
```

这意味着这是一次正式打包发布的新构建,但从可读差异看,目前没有足够证据说明它带来了新的用户界面或新增能力。

### 2. Browser 与 Chrome 插件版本随主应用同步 【实证】

内置 Browser(应用内浏览器)插件与 Chrome(浏览器自动化)插件的 `plugin.json` 版本号均从 `26.602.40724` 更新为 `26.602.71036`:

```diff

```

可感知点是:插件包随主应用一起重新发布。但事实包中没有显示插件描述、权限、脚本入口或能力边界发生变化,因此只能确认版本同步,不能推断新增浏览器能力。


因此,这里能确认的是资源包内容发生变化;不能确认这些变化对应具体用户功能。
