# Codex 桌面版 `26.313.41514`(build 1043)

> 第三方变更日志 · 非官方 · 基于官方更新包逆向差异

这是一次非常克制的小版本更新：文件树规模没有变化，最明确的用户可感知变化集中在 macOS(苹果桌面操作系统) 蓝牙权限描述的调整。  
English one-liner: A small desktop update with a Bluetooth permission wording change and minimal bundle-level churn.

## 关于这份报告

这不是官方变更日志，而是一份**基于官方分发产物的逆向差异报告**。本次属于历史回填：两个版本均已超出官方 Sparkle(应用自动更新框架) 增量窗口，因此没有使用增量包重建流程，而是下载两个官方全量包后直接做结构化 bundle(应用包) 差异。

对比范围是 `26.313.41036`(build 1032，2026-03-16) 到 `26.313.41514`(build 1043，2026-03-17)，平台为 macOS arm64(苹果芯片架构)。

可信度方面，两个官方全量包均给出了 SHA-256(文件指纹)；公证验证结果均为 `CSSMERR_TP_CERT_REVOKED`。事实包说明这类早期版本可能因签名证书后来被吊销而出现该结果，它影响系统安全安装判断，但不影响对 bundle 内容做差异分析。

本文使用两档证据等级：

- **【实证】**：来自 Info.plist(应用元信息文件) 键值、依赖清单、签名 / 公证结果等人类可读直接证据。
- **【信号】**：只来自前端模块名、二进制体积变化等间接证据；这类内容只说明“指向某方向”，无法从产物还原到源码级意图。

## ✨ 重点变化

### 1. 蓝牙权限描述从 Always 切到 Peripheral 【实证】

最新版移除了 `NSBluetoothAlwaysUsageDescription`，新增 `NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'`。

这意味着应用在 macOS 权限声明层面不再使用“Always(始终)”这一键，而改为声明蓝牙外设访问用途。可感知点是：如果系统弹出蓝牙权限相关提示，底层触发的 Info.plist 权限键已经发生变化。

关键证据：

```diff
- /NSBluetoothAlwaysUsageDescription = 'This app needs access to Bluetooth'
+ /NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'
```

### 2. 版本号与构建号正常推进 【实证】

应用短版本从 `26.313.41036` 升至 `26.313.41514`，构建号从 `1032` 升至 `1043`。

这确认本次是 2026-03-16 到 2026-03-17 之间的相邻小版本变化，而不是大规模重打包或渠道切换。

关键证据：

```diff
CFBundleShortVersionString: 26.313.41036 → 26.313.41514
CFBundleVersion: 1032 → 1043
```

### 3. bundle 文件树规模没有扩张 【实证】

外层 bundle 文件数保持 `415 → 415`，新增 `0`、移除 `0`、内容改动 `40`。

这意味着从应用包文件层面看，本版没有引入新的外层组件，也没有移除既有组件；变化主要发生在既有文件内容中。

### 4. app.asar 内容有变化，但不能解读为功能增删 【实证】

`Contents/Resources/app.asar` 的完整性哈希发生变化，说明应用层打包内容确实更新过；但事实包同时标注，`webview/assets` 下大量带 hash 的文件名增减属于 Vite(前端构建工具) 构建产物重命名噪音。

因此，本报告不把 app.asar(应用资源归档) 内部 `新增 13 / 移除 13 / 改动 8` 解释为新增或删除了 13 个真实功能。

关键证据：

```diff
ElectronAsarIntegrity/Resources/app.asar/hash:
93776ba6e64631516c7cc59051e5a6c302acc9aac598443da451de829c5afcaf
→ bcedb0bdad26957b87b9d7199fff261e4f489852d9b29e23528dd6d8419a88a2
```
