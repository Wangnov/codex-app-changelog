本报告基于两份官方签名分发产物的逆向 diff：macOS 使用 Sparkle 全量包对比，Windows 使用 MSIX 分发包对比。报告只引用事实包中出现的文件树、清单、体积、签名校验与 app.asar 统计，不推断未给出的源码意图。

证据等级分两类：【实证】表示来自可读清单、文件树、依赖清单、app.asar 统计或签名校验等直接证据；【信号】表示仅由前端模块名或体积变化提示方向，不能还原源码事实。本批次可归纳的主要变化均为【实证】。

平台归属标记含义：[macOS+Windows] 表示两平台均出现；[仅 macOS] 表示只在 macOS diff 中出现；[仅 Windows] 表示只在 Windows diff 中出现。
