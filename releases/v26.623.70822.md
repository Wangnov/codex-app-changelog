# Codex 桌面版 26.623.70822 macOS / Windows 对照

本批次显示出一个典型的跨平台错位：Windows 包主要是二进制与插件 manifest 级别更新，macOS 包内则同时出现大量前端路由、设置页与 UI 文案信号。  
One-liner: this batch confirms bundled plugin version bumps on both platforms, while macOS exposes broader UI and workflow signals in the packaged frontend.

## 关于这份报告

本文只使用输入事实包中的机器提取事实，覆盖 macOS arm64 `26.623.61825 build 4548 → 26.623.70822 build 4559` 与 Windows x64 `26.623.8305.0 → 26.623.9142.0`。两个平台版本号和发布时间口径不一致，因此下文分别引用各自事实，不把一个平台的结论套到另一个平台。
