## About This Report

Version numbers and release timing may be skewed between platforms, so each conclusion cites the facts for its own platform.

Evidence is split into two levels. `[Confirmed]` comes from plist, manifest, plugin config, dependency lists, readable strings, signing verification, package metadata, or explicit file diffs. `[Signal]` comes from same-name frontend modules, compressed bundle string differences, class names, and size direction. Signals can indicate UI or interaction direction, but they are not source-level facts.

Platform scope is marked as `[macOS+Windows]`, `[macOS only]`, or `[Windows only]`. Hash renames under `webview/assets` and `.vite/build` are treated as build noise per the input and are not counted as functional additions or removals.
