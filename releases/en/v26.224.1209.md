### 4. Build and Code-Quality Tooling Moves from Prettier to oxfmt, with Layered oxlint [Confirmed]

`package.json` shows formatting commands moving from `prettier --check .` / `prettier --write .` to `oxfmt --check` / `oxfmt --write`. The lint script is also split into `lint:oxlint` and `lint:eslint`.

This is not directly user-facing, but it clearly affects the desktop app’s development workflow: formatting and static analysis now lean on `oxfmt` / `oxlint`, while ESLint remains in the chain.

Key diff:
