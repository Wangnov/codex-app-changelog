## ✨ Key Changes

### 1. Build and Test Flows Now Explicitly Rebuild SQLite [Confirmed]

Multiple scripts in `package.json` were changed to run `pnpm run rebuild:sqlite` first, covering `dev`, `forge:package`, `forge:make`, `forge:publish`, `test`, `test:quiet`, and `playwright:agent:repl`.

This means this release places more emphasis on ensuring that the native SQLite module is usable before development, testing, packaging, and publishing. This may not be directly visible to ordinary users, but it is a clear change for development build stability and the automated testing pipeline.

Key evidence:

```diff
-    "test": "node ./scripts/ensure-electron-binary.mjs && vitest run",
+    "test": "pnpm run rebuild:sqlite && node ./scripts/ensure-electron-binary.mjs && vitest run",
```

### 2. TypeScript Compilation Command Switched from `tsc` to `tsgo` [Confirmed]

The `compile` and `tsc` scripts were changed from directly invoking `tsc` to running through `pnpm exec tsgo`.

This shows that the project's internal type-checking / compilation entry point has been adjusted. The fact package does not provide further context for `tsgo`, so only the command-level change can be confirmed here; no inference is made about performance or behavioral differences.

Key evidence:

```diff
-    "compile": "tsc -b",
-    "tsc": "tsc -noEmit",
+    "compile": "pnpm exec tsgo -b",
+    "tsc": "pnpm exec tsgo --noEmit",
```

### 3. Bluetooth Permission Declaration Key Replaced [Confirmed]

`Info.plist` added `NSBluetoothPeripheralUsageDescription` while removing the previous `NSBluetoothAlwaysUsageDescription`; both strings point to a Bluetooth access requirement.

This means the macOS permission declaration key name has changed. The fact package does not show any newly added Bluetooth functionality, so only the migration of permission metadata from one Bluetooth description key to another can be confirmed.

Key evidence:

```text
Added /NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'
Removed /NSBluetoothAlwaysUsageDescription = 'This app needs access to Bluetooth'
```
