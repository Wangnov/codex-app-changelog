## About This Report

This report uses only the machine-extracted facts in the provided fact packages. The macOS comparison covers 26.623.30605 build 4415 to 26.623.31443 build 4441; the Windows comparison covers 26.623.3245.0 to 26.623.3763.0. Platform scope is marked as [macOS+Windows], [macOS only], or [Windows only].

On macOS, Info.plist shows `ChromiumBaseVersion` moving from 149.0.7827.157 to 149.0.7827.197, and `ChromiumBaseBundleVersion` from 7827.157 to 7827.197. On Windows, `149.0.7827.157.manifest` was replaced by `149.0.7827.197.manifest`, and the Google Docs Reading Mode helper manifest version also moved from 149.0.7827.157 to 149.0.7827.197.
