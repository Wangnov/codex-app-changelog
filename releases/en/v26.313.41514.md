## ✨ Key Changes

### 1. Bluetooth Permission Description Switched from Always to Peripheral [Confirmed]

The latest version removed `NSBluetoothAlwaysUsageDescription` and added `NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'`.

This means that, at the macOS permission declaration level, the app no longer uses the “Always” key and instead declares a Bluetooth peripheral access purpose. The user-visible point is: if the system shows a Bluetooth permission prompt, the underlying Info.plist permission key that triggered it has changed.

Key evidence:

```diff
- /NSBluetoothAlwaysUsageDescription = 'This app needs access to Bluetooth'
+ /NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'
```

### 2. Version and Build Numbers Advanced Normally [Confirmed]

The app short version increased from `26.313.41036` to `26.313.41514`, and the build number increased from `1032` to `1043`.

This confirms that this is an adjacent minor-version change between 2026-03-16 and 2026-03-17, rather than a large-scale repackaging or channel switch.

Key evidence:

### 3. The Bundle File Tree Did Not Expand [Confirmed]

The outer bundle file count stayed at `415 → 415`, with `0` added files, `0` removed files, and `40` content changes.

This means that, from the application bundle file level, this release did not introduce new outer components or remove existing ones; the changes are mainly within existing file contents.
