This is not an official OpenAI changelog, but a **reverse-engineered diff report based on officially distributed artifacts**.

### 2. Bluetooth permission declaration key was adjusted [Confirmed]

The macOS permission declaration changed from `NSBluetoothPeripheralUsageDescription` to `NSBluetoothAlwaysUsageDescription`. This indicates the app still declares that it “needs access to Bluetooth,” but the configuration key has changed.

User-visible implication: if the system triggers the related permission prompt, the permission description embedded in the app bundle remains `This app needs access to Bluetooth`, but the new version uses a different Info.plist key.

Key evidence:

```diff

- /NSBluetoothPeripheralUsageDescription = "This app needs access to Bluetooth"
```
