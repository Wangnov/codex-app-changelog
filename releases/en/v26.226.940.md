## About This Report

This is not an official changelog. It is a **reverse-diff report based on official distribution artifacts**. The comparison covers macOS arm64 `26.224.1209` build `697` against `26.226.940` build `720`.

The evidence comes from the bundle file tree, `Info.plist`, `package.json` diffs, newly added readable strings in main/frontend bundles, and frontend module changes after filtering Vite hash noise. Additions and removals under `webview/assets` and `.vite/build` caused by hashed filenames are not treated as feature additions or removals.

There are two evidence levels: **[Confirmed]** means the artifact contains direct human-readable evidence, such as plist keys, config diffs, dependency declarations, or readable strings. **[Signal]** means the evidence is indirect, such as module names, same-name frontend module string diffs, or size changes; it points in a direction, but cannot reconstruct source-level intent.
