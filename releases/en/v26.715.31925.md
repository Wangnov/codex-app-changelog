# Codex Desktop 26.715.31925 macOS / Windows Comparison

> Third-party cross-platform changelog · Unofficial

The shared confirmed change is a synchronized version update for the bundled Browser and Chrome plugins. The more visible product signals center on task onboarding, plugin sharing, temporary chat, and desktop-app entry points on Windows, versus debug, settings, and workbook changes on macOS.

Browser plugin manifests moved together, while Windows shows broader task, sharing, temporary-chat, and desktop-control signals and macOS concentrates on debug, settings, and workbook changes.

## About This Report

This report compares machine-extracted differences between two full official distribution packages. The macOS comparison covers `26.715.31251` build 5538 through `26.715.31925` build 5551. The Windows comparison covers MSIX `26.715.3651.0` through `26.715.4045.0`. The platform versions are not aligned, so every conclusion is based only on the corresponding platform fact package.

Evidence uses two levels:

- [Confirmed] covers plist fields, plugin manifests, dependency lists, file trees, signature verification, and other directly readable configuration.
- [Signal] covers frontend module names, string or style-set changes inside modified same-name minified modules, and size changes. It shows a direction entered the package, but does not prove availability or constitute source-level fact.

Platform labels are `[macOS+Windows]`, `[macOS only]`, and `[Windows only]`.

## ✨ Highlights

### Browser and Chrome Plugin Baseline Refresh

[Confirmed] [macOS+Windows]

On both platforms, the bundled `browser` and `chrome` plugin manifests changed their plugin versions from `26.715.31251` to `26.715.31925`.

The existing `browser` description still targets the ChatGPT in-app browser, local web targets, `localhost`, loopback addresses, `file://`, clicking, typing, and screenshots. The `chrome` description still targets automation dependent on existing Chrome tabs, authenticated sessions, cookies, extensions, and ChatGPT Chrome Extension state. The supplied diffs confirm version synchronization but show no description change.

The outer Windows MSIX version moved from `26.715.3651.0` to `26.715.4045.0`, while these in-package plugins use `26.715.31925`. The MSIX version must not be substituted for the manifest version.

### Windows Computer Use Plugin Version Update

[Confirmed] [Windows only]

The Windows `computer-use` plugin manifest also moved from `26.715.31251` to `26.715.31925`. Its description explicitly states that it controls Windows desktop apps from ChatGPT through Computer Use.

This was not a runtime expansion: Windows `cua_node` stayed at `287.5 MB`, with zero new dependencies. The macOS runtime likewise stayed at `365.7 MB` with zero new dependencies, but the macOS fact package contains no corresponding `computer-use` manifest diff.
