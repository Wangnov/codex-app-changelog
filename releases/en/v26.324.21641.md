## ✨ Highlights

### 1. Bluetooth Permission Description Key Changed from Always to Peripheral [Confirmed]

`/NSBluetoothAlwaysUsageDescription` was removed from `Info.plist`, and `/NSBluetoothPeripheralUsageDescription` was added. Both use the same text: `This app needs access to Bluetooth`.

This means the macOS permission key used by the app to declare its Bluetooth access purpose has changed. The user-visible effect may appear in system permission prompts or permission review paths, but the fact package does not provide runtime behavior, so no further inference can be made about changes to the Bluetooth functionality itself.

Key evidence:

```diff
- /NSBluetoothAlwaysUsageDescription = 'This app needs access to Bluetooth'
+ /NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'
```

### 2. Version and Build Number Updated [Confirmed]
