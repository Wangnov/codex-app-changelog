## ✨ Key Changes

### 1. Bluetooth Permission Description Migrated to the New macOS Privacy Key [Confirmed]

`Info.plist` adds `NSBluetoothAlwaysUsageDescription` with the value `This app needs access to Bluetooth`; at the same time, it removes the old `NSBluetoothPeripheralUsageDescription`, whose previous value was also `This app needs access to Bluetooth`.

This means the system description key for Bluetooth permission changed. The user-visible impact is mainly at the macOS permission prompt or system privacy-permission metadata layer; the fact package provides no further evidence that Bluetooth-related functionality itself changed.

Key evidence:

```diff
+ /NSBluetoothAlwaysUsageDescription = 'This app needs access to Bluetooth'
- /NSBluetoothPeripheralUsageDescription = 'This app needs access to Bluetooth'
```

### 2. App Version Bumped to build 1100 [Confirmed]

The app short version changed from `26.317.21539` to `26.318.11754`, and the build changed from `1088` to `1100`.

This is a clear version bump, but based on the current fact package, apart from the permission-key migration and build artifact content changes, there is not enough direct evidence to confirm any new user-visible functionality.
