# Codex Desktop 26.623.31921 macOS / Windows Comparison

> Third-party cross-platform changelog · Unofficial

This batch shows macOS moving from 26.623.31443 to 26.623.31921, while Windows moves from 26.623.3763.0 to 26.623.4041.0. The two packages do not expose identical versioning, chunking, or UI signals, so conclusions must stay platform-scoped.

One-liner: Codex Desktop 26.623.31921 lines up bundled browser plugins while macOS moves toward usage/runtime/model settings, and Windows surfaces plugin sharing, AppGen sharing, debug, RPC and mention-entry signals.

## About This Report

This report uses only the machine-extracted facts in the provided packages: version and channel metadata, bundle file trees, bundled plugin manifests, Computer Use runtime totals, cua_node dependencies, app.asar summaries, frontend module additions/removals, same-name frontend module clues, readable strings, Info.plist or manifest-adjacent changes, and concrete file diffs.
