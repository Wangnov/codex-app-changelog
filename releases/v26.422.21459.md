# Codex 桌面版 `26.422.21459` build 2049

> 第三方变更日志 · 非官方 · 基于官方更新包逆向差异

本版从 `26.422.20832` 升至 `26.422.21459`,bundle 文件数量保持不变,可读变化主要指向 Web preview 相关界面的命名与资源卡展示调整。  
One-liner: a small desktop update with visible signals around web preview resource-card presentation.

## 关于这份报告

这不是官方变更日志,而是一份**基于官方分发产物的逆向差异报告**。本次为历史回填,官方 appcast 未对 `26.422.20832` → `26.422.21459` 提供增量包,因此改为下载两个官方全量包后做结构化 bundle 对比。

证据分两档: **【实证】**表示来自 Info.plist、签名、公证、哈希、依赖清单等人类可读的直接证据;**【信号】**表示只来自前端模块名、同名模块字符串差异或体积变化,只能说明它们指向某方向,但无法从产物还原到源码级意图。

对比范围是 macOS arm64:上一版 `26.422.20832` build `2025`,发布于 `2026-04-23`;最新版 `26.422.21459` build `2049`,发布于 `2026-04-24`。

## ✨ 重点变化

### 1. Web preview 的资源卡标题与展示文案有调整 【信号】

同名前端入口模块 `index` 的可读字符串发生变化,新增了 `Title for a generated website preview resource card`、`Web preview`、`localConversation.endResource.websiteTitle`,同时移除了 `Open web preview`。

这指向 Web preview 在会话资源卡或生成网站预览展示上的文案与标题处理有调整,但该证据只来自构建后的前端模块字符串差异,无法从产物还原到源码级意图。

### 2. 应用版本与 app.asar 完整性哈希更新 【实证】

Info.plist 显示应用短版本号从 `26.422.20832` 更新到 `26.422.21459`,build 从 `2025` 更新到 `2049`。同时 `ElectronAsarIntegrity` 中 `Resources/app.asar/hash` 也发生变化。

这意味着桌面 App 的应用层包确实被替换,但从可读差异看,本版没有新增 bundle 文件、移除 bundle 文件,也没有新增 cua_node 依赖。

### 3. Bundle 结构基本稳定,变化集中在少量内容文件 【实证】

bundle 文件数保持 `507 → 507`,新增 `0`、移除 `0`,内容改动 `46`。体积变化最大的可见组件是 `Contents/Resources/app.asar`,变化为 `+0.01 MB`;`Contents/Resources/codex_chronicle` 变化为 `+0.00 MB`。

这意味着本版更像是一次小幅应用层更新,而不是一次增加大量原生组件或资源文件的版本。
