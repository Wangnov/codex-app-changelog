## 关于这份报告

这不是官方变更日志，而是一份**基于官方分发产物的逆向差异报告**。本版属于历史回填：两个版本均超出官方 Sparkle(应用自动更新框架)增量窗口，因此本次没有使用增量包重建流程，而是直接下载两个官方全量包进行结构化 bundle(应用包目录)差异比较。

可信度方面，本次信任根来自 Apple(苹果)公证与 OpenAI Developer ID(开发者身份签名)。上一版与最新版均通过 `spctl` 公证验证，结果为 `accepted`，来源均为 `Notarized Developer ID`，签名主体均为 `Developer ID Application: OpenAI OpCo, LLC (2DC432GLL2)`。

本文使用两档证据等级：

- **【实证】**：来自人类可读的直接证据，例如 `Info.plist` 键值、签名与公证验证结果、依赖清单或结构化差异。
- **【信号】**：来自间接证据，例如前端模块名或二进制体积变化。此类变化只指向某方向，但无法从产物还原到源码级意图。
