## 关于这份报告

这不是官方变更日志，而是一份**基于官方分发产物的逆向差异报告**。本次对比范围是 macOS arm64 上的 `26.422.21637` build 2056 与 `26.422.30944` build 2080，二者发布日期均为 2026-04-24。

本版为历史回填：官方 appcast 未对这一对版本提供增量包，实际方法是下载两个官方全量包后直接比较，并结合结构化 bundle 差异归纳变化。

证据等级分两档：

- **【实证】**：来自 Info.plist、签名/公证输出、文件树统计、依赖清单、可读结构化差异等直接证据。
- **【信号】**：仅来自前端模块名、二进制体积变化等间接线索，只能说明指向某方向，但无法从产物还原到源码级意图。

本报告会主动忽略 `webview/assets`、`.vite/build` 下 hash 文件名变化导致的构建噪音，不把这类增减写成新增或删除功能。
